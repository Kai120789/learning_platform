# 2. Границы

Здесь описано, какая логика реализуется приложением, а какая - нет

## Что входит
- Асинхронная обработка пользовательских действий:
  - отправка домашних заданий на проверку
  - прохождение теоретических уроков и тестов
  - отправка вебинаров на загрузку
  - автоматическая проверка тестов
- Управление учебными программами и заданиями
- Планирование расписаний и задание времени напоминаний
- Фоновая обработка медиа-контента (видео, вебинары)
- Публикация уведомлений пользователям через события
- Создание и управление аккаунтами преподавателей (для админа)

## Что не входит
- Система оплаты или подписки
- Полноценная админка с ролями
- Рекомендательные алгоритмы
- Интеграции с внешними LMS-системами
- Мобильные приложения
